# Sport-Tek ST550 front mockup images

Place front-view product photos in this folder using the naming pattern:

`{variant-id}-front.jpg`

Examples:

- `atomic-blue-front.jpg`
- `true-navy-front.jpg`
- `white-front.jpg`

The preview tool loads `.jpg` first. If a JPG is missing, it falls back to the matching `.svg` placeholder in this folder.

Variant IDs:

- atomic-blue
- black
- carolina-blue
- deep-orange
- deep-red
- forest-green
- gold
- iron-grey
- kelly-green
- maroon
- purple
- royal
- silver
- true-navy
- true-red
- white
